Daily Challenge Review

How to use:
1. Run npm run generate-daily to create candidate challenges.
2. Open this file in your browser (File → Open).
3. Click "Load manifest.json" and select public/data/daily/manifest.json.
4. Review crops. Click on the reveal image to set the insect center — a green crosshair shows where crops will be centered.
5. After adjusting centers, click "Download manifest" and replace the original.
6. Run npm run recrop-daily to regenerate crops with corrected centers.
7. Reload manifest to verify. Approve good challenges, reject bad ones.
8. Download final manifest, commit, and deploy.
