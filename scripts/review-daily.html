Daily Challenge Review

How to use:
1. Run npm run generate-daily to create candidate challenges.
2. Open this file in your browser (File → Open).
3. Click "Load manifest.json" and select public/data/daily/manifest.json.
4. Review each challenge's crops. Approve or reject.
5. Click "Download approved manifest" to save. Replace the original file with it.
6. Commit and deploy.
